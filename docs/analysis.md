# Análisis del sistema

## Funcionalidades básicas

- Crear alarma
- Eliminar alarma
- Activar alarma
- Desactivar alarma
- Configurar hora y minuto
- Configurar etiqueta
- Configurar sonido y volumen
- Repetición semanal
- Posponer alarma
- Detener alarma
- Consultar próximas alarmas

## Funcionalidades avanzadas

### 1. Modo circadiano
Incremento gradual del volumen al sonar la alarma.

### 2. Detector de conflictos
Detecta alarmas configuradas con menos de 5 minutos de diferencia.

### 3. Reto matemático
La alarma solo puede detenerse tras resolver correctamente una operación matemática.
